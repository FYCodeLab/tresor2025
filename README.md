## 💻 What was used to build it?

- **Language**: JavaScript and TypeScript (a safer version of JavaScript).  
- **Framework**: React (a popular library for building interactive websites).  
- **Styling**: Tailwind CSS, with components provided by Lovable.

## 📍 What is GitHub Pages?

GitHub Pages is a free service from GitHub that allows anyone to put a website online directly from a code repository.  
It is widely used by students, teachers, and developers because it is simple and free.
